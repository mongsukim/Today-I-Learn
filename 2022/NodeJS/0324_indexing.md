 ```indexing``` 사용 
 
 ### Binary Search 적용하려면 미리 숫자순 정렬이 되어있어야 함 
 
 index 생성방법
 
 - 몽고db에 들어가서 clusters -> 왼쪽에서 데이터베이스 선택 -> collection -> indexes -> create index

문자순 정렬을 원하면
```js
{
 "제목":"text"
}
```


숫자순 정렬을 원하면
```js
{
 _id:1
}
```
