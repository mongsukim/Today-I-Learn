- 로컬스토리지
 
 로컬스토리지에 저장될땐 문자열로 저장이 된다.  
- Number를 꺼내 쓸땐 ```parseInt``` 를 쓴다.   
- 객체를 꺼내 쓸땐 ```JSON.parse```를 쓴다   
 
 예 : 
```js
const item3 = JSON.parse(localStorage.getItem('item3'));

```
